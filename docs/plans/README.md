# plans

**Structured discovery only.** Contents: interview directories and decision working
documents — material that *precedes* tracked work and becomes append-only history once
its outputs land.

Anti-catch-all routing rule — if it isn't structured discovery or a decision working
doc, it does not go here:

- A decision → `docs/decision-log.md` or an ADR
- Work to do → a tracked issue
- Operational how-to → `docs/runbooks/`
- User-facing content → the project's docs source of truth
- A term → the glossary
